HEADER MOSAIC IMAGES
====================

Drop image files (.jpg .jpeg .png .webp .gif .avif) into THIS folder.

They are picked up automatically by the landing-page header mosaic — no code
change and no rebuild needed. Just add or remove files here and reload the page.

Tips
----
- Portrait images (3:4) fit the tiles best; others are cropped to fit (never
  squashed).
- Filenames are listed alphabetically, so prefix with 01_, 02_, ... if you want
  a specific order.
- If this folder has no images, the mosaic falls back to coloured gradient tiles.

This README and any non-image files are ignored.
